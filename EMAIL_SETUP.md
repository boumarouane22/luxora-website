# إعداد البريد الإلكتروني لموقع Luxora

## الوضع الحالي
- تم إزالة جميع معلومات التواصل باستثناء البريد الإلكتروني
- البريد الإلكتروني المستخدم: **luxora.invest@gmail.com**
- تم إضافة وظيفة إرسال البيانات إلى البريد الإلكتروني

## كيفية تفعيل إرسال البريد الإلكتروني الحقيقي

### الخطوة 1: إنشاء حساب EmailJS
1. اذهب إلى [EmailJS.com](https://www.emailjs.com/)
2. أنشئ حساب جديد مجاني
3. اربط حساب Gmail الخاص بك (luxora.invest@gmail.com)

### الخطوة 2: إعداد الخدمة
1. في لوحة التحكم، اذهب إلى "Email Services"
2. أضف خدمة Gmail جديدة
3. احفظ Service ID

### الخطوة 3: إنشاء قالب البريد
أنشئ قالب جديد في EmailJS باستخدام هذا النموذج:

```
الموضوع: {{subject}}

من: {{from_name}}
البريد الإلكتروني: {{from_email}}
الهاتف: {{phone}}

نوع الطلب: {{form_type}}

{{#if position}}
المنصب المطلوب: {{position}}
سنوات الخبرة: {{experience}}
{{/if}}

{{#if investor_type}}
نوع المستثمر: {{investor_type}}
مبلغ الاستثمار: {{amount}}
فترة الاستثمار: {{period}}
مستوى المخاطر: {{risk_tolerance}}
{{/if}}

الرسالة:
{{message}}

---
تم الإرسال من موقع Luxora
```

### الخطوة 4: تحديث الكود
في ملف `script.js`، استبدل:

```javascript
// Initialize EmailJS
emailjs.init("YOUR_PUBLIC_KEY"); // ضع Public Key الخاص بك هنا
```

وفي دالة `sendEmailNotification`:

```javascript
emailjs.send(
    'YOUR_SERVICE_ID',    // ضع Service ID هنا
    'YOUR_TEMPLATE_ID',   // ضع Template ID هنا
    params,
    'YOUR_PUBLIC_KEY'     // ضع Public Key هنا
).then(resolve, reject);
```

- رقم الهاتف
- المنصب المطلوب
- سنوات الخبرة
- الرسالة الإضافية

### لطلبات الاستثمار:

- البريد الإلكتروني

- مبلغ الاستثمار
- فترة الاستثمار
- مستوى تحمل المخاطر
- الأهداف الاستثمارية

## الوضع الحالي (للاختبار)
- البيانات تُعرض في Console المتصفح
- رسائل النجاح تظهر للمستخدم
- جميع الوظائف تعمل باستثناء الإرسال الفعلي للبريد

## ملاحظات مهمة
- خدمة EmailJS مجانية حتى 200 بريد شهرياً
- يمكن ترقية الحساب للحصول على المزيد
- البيانات الحساسة محمية ولا تُحفظ في المتصفح
- جميع عمليات التحقق من البيانات تعمل بشكل طبيعي

## اختبار الموقع
1. افتح `index.html` في المتصفح
2. اذهب إلى قسم "انضم إلينا"
3. املأ أي من النماذج
4. اضغط إرسال
5. تحقق من Console المتصفح لرؤية البيانات المرسلة
